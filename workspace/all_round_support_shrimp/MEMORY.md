### Self-Improving Proactive Agent
- **技能目录**: `/Users/zhu/.openclaw/workspace/skills/self-improving-proactive-agent`
- **核心架构**: 一技两件套——Self-improving(持续学习) + Proactivity(主动推进)

#### 文件布局
```
~/self-improving/       # 持久化学习记忆
├── memory.md           # HOT: 确认过的规则和偏好
├── corrections.md      # 近期修正和待推广的经验
└── archive/            # 冷存储

~/proactivity/          # 活跃任务状态
├── session-state.md    # 当前目标/决策/阻塞/下一步
├── heartbeat.md        # 轻量级循环跟进
├── patterns.md         # 可复用的主动经验
└── memory/
    └── working-buffer.md  # 脆弱临时上下文
```

#### 6 大核心原则
1. **只从显式证据学习** - 用户修正、明确偏好、重复成功工作流;不推断自沉默或猜测
2. **推动下一个有用步骤** - 发现缺失步骤、阻塞点、准备选项;信号弱时保持安静
3. **信息路由到正确位置** - 持久学习→self-improving, 活跃状态→session-state, 临时缓存→working-buffer
4. **先恢复再提问** - 按顺序读 HOT 记忆→stable memory→session state→buffer→只问缺失部分
5. **验证实现而非意图** - 改变真实机制而不仅是文字;测试结果后再报告成功
6. **严格边界内的主动性** - 发消息/花钱/删除数据/公开动作/承诺他人前必须询问

#### 边界红线 ⚠️
- ✅ 可以自动执行：内部检查、草案提案、本地状态更新、只读验证、可撤销准备
- ❌ 必须事先询问：发消息联系他人、花钱、删除数据、公开发布、做承诺、代排日程
- 🚫 永远不做：从沉默推断偏好、假装确定、泄露共享频道私有上下文、在记忆中存凭证密码

#### 学习规则
- **有效输入**: 显式修正、明确持久偏好、有意义工作后的反思、重复成功工作流
- **无效输入**: 沉默、一次性指令、对意图的猜测
- **Promotion**: 7 天内重复 3x → 提升到 HOT memory; 未使用 30 天→降为 WARM;90 天→归档
- **反思格式**: `CONTEXT: [任务类型] / REFLECTION: [发生了什么] / LESSON: [下次怎么做]`

#### Heartbeat 行为规范
- **该检查**: 已承诺的跟进、陈旧的阻塞点、即将截止的 Deadline、无下一步的活跃工作、维护机会
- **该发消息**: 有变化/需决策解阻塞/准备好草案建议/等待有实际成本
- **应保持沉默**: 项未变/信号弱/消息只是重复旧信息/行动仍模糊

### Skill Finder (技能查找器)
- **技能目录**: `/Users/zhu/.openclaw/workspace/skills/skill-finder-cn`
- **触发词**: 找 skill、find skill、搜索 skill、有什么 skill 可以...
- **作用**: 帮助用户发现和安装 ClawHub 上的 Skills

#### 标准工作流程
```
1. 理解用户需求 → 2. 提取关键词 → 3. clawhub search "关键词"
4. 列出相关 Skills(含 stats) → 5. 推荐最合适的 → 6. 安装并验证
```

#### 核心命令
```bash
# 搜索技能
clawhub search "时间管理" / clawhub search "crypto price monitor"

# 查看详情 + 统计数据
clawhub inspect <skill-name>
# 或 API 查询 stats: curl "https://clawhub.ai/api/v1/skills/<name>" | jq '.skill.stats'

# 安装技能
clawhub install <skill-name>

# 验证安装 (新增！关键步骤)
ls ~/.openclaw/workspace/skills/<skill-name>/SKILL.md

# 检查已安装的 Skills
clawhub list
# 或 ls ~/.openclaw/workspace/skills/
```

#### 输出格式规范
搜索结果应包含：
- ✅ Skill 名称
- ✅ 中文描述
- ✅ 下载量 & Stars 数
- ✅ 是否已安装状态 (✅已安装/❌未安装)
- ✅ 推荐安装提示和安装命令
