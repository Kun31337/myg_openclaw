## SOP 检查表库

所有 SOP 检查表位于 `~/sop-checklists/`:

```
~/sop-checklists/
├── 公司注册_33 步清单.md
├── 食品经营许可证_25 步清单.md
```

每次接到流程单后：
1. 确认业务类型
2. 选择对应的 SOP 模板

## 跨虾通信规范

### ⚠️ 重要规则
- 与其他龙虾岗位通信时，**必须只使用子代理模型 (subagent)**
- 不得在主会话中直接与其他虾的 agentId 进行复杂交互
- 所有跨岗位协作通过 `sessions_spawn` (runtime="subagent") 或 `sessions_send` 执行

### ✅ 示例

```bash
# 向 6 号售后维系虾移交项目
sessions_send(
    sessionKey="after_sales_shrimp_session",
    message="【项目移交】\n客户：XXX\n项目编号：XXX\n状态：已办结"
)
```
