```markdown
# 虾系运营助手 - 周期性自动检查项

```

## 📊 每日检查项 (Daily)

### 1. 视频数据追踪 ⭐️
- [ ] 查看昨日发布视频的播放量、完播率
- [ ] 统计评论数和互动率
- [ ] 记录转化咨询数量

### 2. 评论区互动
- [ ] 回复未读评论 (需 100% 回复率)
- [ ] 标记高意向客户线索
- [ ] @私域运营虾 (3+号) 传递优质线索

### 3. 热点话题监控
- [ ] 搜索 "ICP 证"、"EDI 证" 相关热搜
- [ ] 关注政策类新闻更新
- [ ] 记录可蹭的热点关键词

---

## 📈 每周检查项 (Weekly)

### 周一 10:00
- [ ] 制定本周选题计划 (至少 3 条)
- [ ] 同步给品牌展示虾 (1 号) 确认包装风格
- [ ] 预约拍摄时间

### 周五 18:00
- [ ] 输出周报：数据统计 + 问题分析
- [ ] 向 7 号数据虾请求详细分析报告
- [ ] 规划下周改进方向

## 🎯 月度检查项 (Monthly)

### 月末最后一天
- [ ] 复盘月度 KPI 完成情况
- [ ] 更新脚本模板库
- [ ] 分析爆款视频共性
- [ ] 向 0 号龙虾老板汇报月度总结

## 🔔 触发式检查

### 当出现以下情况时立即响应:
- [ ] 某条视频播放量异常 (>1 万) → 分析原因并跟进评论
- [ ] 完播率骤降 (<30%) → 调整内容方向
- [ ] 收到大量同类咨询 → 制作专题视频回应
- [ ] 发现政策新规 → 快速产出解读内容
