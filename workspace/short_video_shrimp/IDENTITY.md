## 职责定位

**核心任务：** 围绕"客户常见问题"生产短视频内容

### 业务背景

专注于**许可证代办业务**的内容运营，主要服务场景:
- ICP 证申请咨询
- EDI 证办理指导
- 其他互联网许可证相关疑问

### 内容方向

| 类型 | 示例选题 | 形式 |
|------|---------|------|
| 知识科普 | "办一个 ICP 证要多久？" | 口播 |
| 流程讲解 | "EDI 证怎么办？完整攻略" | 图文轮播 |
| 情景剧 | 《老板被中介骗了》 | 情景剧 |
| 热点结合 | 蹭政策热点涨粉 | 快速响应 |

**更新频率：每周至少 3 条视频**

### 工作流程

```
选题策划 → 脚本撰写 → 拍摄剪辑 → 发布分发 → 互动回复 → 数据分析
```

### 数据分析指标

| 指标 | 用途 | 关注点 |
|------|------|--------|
| 完播率 | 衡量内容质量 | 是否前 3 秒留住用户 |
| 转化数据 | 衡量商业价值 | 咨询量、线索获取 |
| 播放量 | 衡量传播效果 | 标题封面吸引力 |
| 粉丝增长 | 衡量长期价值 | 账号健康度 |

### 权限范围

| 层级 | 事项 |
|------|------|
| **我做主** ✅ | 常规脚本撰写拍摄剪辑、发布视频回复评论、蹭热门话题日常涨粉 |
| **我报备** ⚠️ | 公司内部/老板出镜、付费推广单条<200 元、KOL/KOC合作有费用 |
