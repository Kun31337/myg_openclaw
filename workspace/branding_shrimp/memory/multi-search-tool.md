# Multi Search Engine Simple - 联网搜索专用技能

## 技能路径

`/Users/zhu/.openclaw/workspace/skills/multi-search-engine-simple/SKILL.md`

## 功能说明

集成 **10 个免费搜索引擎**，支持国内主流搜索平台：

### 支持的平台（10 个）

| 引擎 | 链接 |
|------|------|
| Baidu | `https://www.baidu.com/s?wd={keyword}` |
| Bing CN (中国) | `https://cn.bing.com/search?q={keyword}&ensearch=0` |
| Bing INT (国际) | `https://cn.bing.com/search?q={keyword}&ensearch=1` |
| 360 | `https://www.so.com/s?q={keyword}` |
| Sogou | `https://sogou.com/web?query={keyword}` |
| WeChat (微信搜一搜) | `https://wx.sogou.com/weixin?type=2&query={keyword}` |
| Toutiao (今日头条) | `https://so.toutiao.com/search?keyword={keyword}` |
| Jisilu | `https://www.jisilu.cn/explore/?keyword={keyword}` |
| Ecosia | `https://www.ecosia.org/search?q={keyword}` |
| WolframAlpha | `https://www.wolframalpha.com/input?i={keyword}` |
